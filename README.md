# warehouse-vue

Projeto criado para consumir dados da API rails [warehouse-app](https://github.com/cellaaleo/warehouse-app)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
